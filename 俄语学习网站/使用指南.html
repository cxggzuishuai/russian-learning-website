零代码操作，轻松管理你的俄语学习网站

🚀 如何打开网站

1 找到文件夹 俄语学习网站
2 双击 index.html 文件，浏览器会自动打开网站
就这么简单！不需要安装任何软件，不需要启动服务器。

📦 资源上架 / 下架

进入管理后台

点击网站右上角的红色 「管理后台」 按钮，会弹出管理面板。

添加新资源（上架）

1 在管理面板中，确保选中 「📦 资源管理」 标签页
2 点击右上角蓝色 「+ 添加资源」 按钮
3 填写资源信息（带 * 的必填）：
字段说明示例
资源名称（中文）*资源的中文标题俄语日常对话100句
资源名称（俄文）俄文标题，可选100 фраз на каждый день
分类 *下拉选择所属分类词汇积累
难度级别入门 / 中级 / 高级入门
资源类型文章 / 视频 / 音频 / 闪卡文章
资源链接点击后跳转的网址https://example.com/lesson1
资源描述简要介绍资源内容精选100句日常用语...
缩略图URL封面图片网址（可选）留空用默认图标
标签逗号分隔的关键词日常, 对话, 入门
上架状态开关控制是否显示打开 = 上架
首页推荐是否显示在首页精选区打开 = 推荐
4 点击 「保存」，资源立刻生效

下架资源

在资源列表中，找到要下架的资源，点击红色 「下架」 按钮即可。下架后资源不会被删除，可随时重新上架。

重新上架

找到已下架的资源（可通过筛选「已下架」状态查看），点击绿色 「上架」 按钮。

编辑资源

点击资源行的 「编辑」 按钮，修改信息后保存。

删除资源

点击 「删除」 按钮（请谨慎，删除后不可恢复）。

搜索和筛选

在管理面板顶部有搜索框和筛选器，可按关键词搜索、按分类筛选、按上架/下架状态筛选。

🎨 首页布局调整（拖拉排序）

1 进入管理后台，点击 「🎨 首页布局」 标签页
2 你会看到所有首页板块的列表，每个板块前面有 ⠿ 拖拽手柄
3 拖拽排序：按住板块不放，上下拖动到想要的位置，松开即保存
4 显示/隐藏：每个板块右侧有开关，关掉后该板块不显示在首页
修改实时生效！关闭管理面板后首页会自动刷新显示新布局。

⚙️ 网站设置

在管理后台点击 「⚙️ 网站设置」 标签页，可以修改：

  • 网站标题：显示在导航栏的名称
  • 网站副标题：Hero区域的描述文字
  • Hero 主标题：大横幅的主要文字
  • Hero 俄文副标题：大横幅的俄文文字

💾 数据备份与恢复

⚠️ 重要：所有修改保存在浏览器的本地存储中。如果清除浏览器数据，修改会丢失！请定期备份。

导出备份

在「网站设置」页面，点击 「📦 导出全部数据」，会下载一个 JSON 文件，包含所有资源和布局设置。

导入数据

点击 「📥 导入数据」，选择之前导出的 JSON 文件，数据会自动恢复。

恢复默认

点击 「🔄 恢复默认数据」 可以将所有数据恢复到初始状态（所有修改会丢失）。

❓ 常见问题

Q: 修改后为什么刷新页面数据还在？

A: 数据保存在浏览器的「本地存储」中，刷新不会丢失。只有清除浏览器数据或点击「恢复默认」才会重置。

Q: 换了电脑怎么办？

A: 先在原电脑「导出全部数据」，把 JSON 文件拷到新电脑，再「导入数据」即可。

Q: 资源链接填什么？

A: 填写资源的实际网址（如视频链接、文章链接等）。如果暂时没有，填 # 即可。

Q: 如何部署到互联网上？

A: 将整个「俄语学习网站」文件夹上传到任意网页托管服务（如 GitHub Pages、Vercel、Netlify），即可通过网址访问。这些服务都是免费的。
